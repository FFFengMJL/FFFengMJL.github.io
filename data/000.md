今天发现了个 OBS 录播小技巧：

在 OBS 使用录制浏览器录制腾讯课堂时，可以通过设置自定义 CSS ，来将 `XXX 正在观看直播`的悬浮条屏蔽掉

通过 __F12__ 查看悬浮所在的元素 *id* 为 marquee

![marquee](./marquee.png)

具体操作是在 `自定义 CSS` 中添加如下代码：

```css
/* marquee 是你想要屏蔽掉的 */
#marquee {
  display: none
}
```

之后刷新网页，就可以达到在 OBS 录制**腾讯课堂**的时候屏蔽掉悬浮条的效果了
