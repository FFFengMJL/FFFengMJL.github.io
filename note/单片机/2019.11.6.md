经典计算机架构：
* 冯诺依曼结构（指令和数据在同一个RAM）
* 哈佛结构（指令和数据在两个RAM）

# 冯诺依曼结构

* 1940S到50S提出
* 顺序执行（串行） -> 成本较低

最早的单片机： The Intel 4004 (1971)
* 16-pin DIP package
* 92,000 instructions/s
* Processor had a small address space for data and instructions
* 只能做基本的+-*
* 只能用汇编和机器码
* 是当时计算器的核心

# 哈佛结构

* 数据和指令并行（高效率）

重点：两者的差别
