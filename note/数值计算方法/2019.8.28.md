## Software

### Tool

- 文字处理软件：Word, LaTex  
- 数据库操作  
- 作图：可以使用 MATLAB  
- 数学相关的编程  

> - 推荐使用正版  
> - Batch commands 批处理命令  
> - Tolerance Error 容差  
> - MATLAB 强大在于画图方便强大  
> - 标量的变量用*斜体*，运算符用正体，标量用**粗黑体**，矩阵用大写  
> - Word 中使用 MathType 工具来书写数学公式  

MATLAB中  
- 矩阵中用`分号`分行，但是最后一行不用分号；`逗号/空格`来分隔元素  
- 常见函数：  
    - 三角函数  
    - 对数函数  
    - 双曲函数  
    - 虚数函数  
- 简单命令：  
    - clc -clears window  
    - clg -clears graphic window  
    - clear -clear the workspace  
    - who -variable list  
    - whos -variable list with size  
    - help  
    - lookfor  -look for *sth*
    - tic ... toc 可用于计算时间复杂度
- 简单符号：  
    - ^C -an escape from loop
    - inf -infinity
    - NaN -No a Number / No numerical value

> Complexity 复杂度：  
> - Operation Complexity  
> - Time Complexity  
> - Space Complexity
